Descripción general

Componente Básico​

El Mini Program proporciona a los desarrolladores una serie de componentes básicos para que los desarrolladores puedan combinarlos para el desarrollo de servicios.

Tipo de Atributo​

El componente proporciona una serie de configuraciones de atributos. Cada valor de atributo tiene un requisito de tipo:

TipoDescripciónNotas
BooleanoBooleano
NúmeroNúmero
CadenaCadena
ArregloArreglo
ObjetoObjeto
EventHandleManejador de EventosNecesita definir la implementación para el manejador de eventos en la página (Page)
anyCualquier tipo

Atributo Común del Componente​

Todos los componentes incluyen los siguientes atributos:

PropiedadTipoDescripciónNotas
idStringIdentificador único del componente.
classStringClase de estilo.
styleStringEstilo en línea.
data-*AnyAtributos personalizados.Cuando se desencadena el evento, el atributo personalizado se transfiere al manejador de eventos.
on catchEventHandleVinculación de eventos, siguiendo las especificaciones de nomenclatura ump, como onTap.Consulte el event.

Consejos​

Se requiere {{}} para transferir datos dentro del tipo de atributo especificado. Por ejemplo:

<view disable-scroll="false"> <!-- Error es una cadena, no un booleano, equivalente al tipo booleano true -->
<view disable-scroll="{{false}}"> <!--correcto, o atributo vacío, significa falso-->
