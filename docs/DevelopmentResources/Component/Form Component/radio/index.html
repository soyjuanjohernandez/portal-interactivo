Radio

radio-group​

Grupo de selectores de Radio.

PropiedadesTipoDescripción
onChangeEventHandleSe activa cuando cambia el elemento seleccionado. event.detail = {value: valor del radio seleccionado}.
nameStringNombre del componente, utilizado para el envío de datos del formulario obtenidos.

radio​

Elemento de radio.

PropiedadesTipoPor DefectoDescripción
valueStringValor del componente, valor transportado en el evento de cambio cuando se selecciona.
checkedBooleanfalseSeleccionado o no en el momento actual.
disabledBooleanfalseDeshabilitado o no.
colorColorColor del radio.

Captura de pantalla​

Código de Ejemplo​

<radio-group class="radio-group" onChange="radioChange">
  <label class="radio" a:for="{{items}}">
    <radio value="{{item.name}}" checked="{{item.checked}}"/>{{item.value}}
  </label>
</radio-group>
Page({
  data: {
    items: [
      {name: 'angular', value: 'AngularJS'},
      {name: 'react', value: 'React', checked: true},
      {name: 'polymer', value: 'Polymer'},
      {name: 'vue', value: 'Vue.js'},
      {name: 'ember', value: 'Ember.js'},
      {name: 'backbone', value: 'Backbone.js'},
    ]
  },
  radioChange: function(e) {
    console.log('Estás seleccionando el framework:', e.detail.value)
  }
})
