# my.showSharePanel

Utilice esta API para activar la compartición.

:::info[NOTA:]
Cuando utilice esta API para activar la compartición, pase el ```código``` de valor al parámetro de solicitud de [page.onShareAppMessage](/portal-interactivo/docs/DevelopmentResources/Framework/Page/mecanismos%20en%20pagina).
:::

## Código de muestra

```js
// .js
my.showSharePanel();
```
