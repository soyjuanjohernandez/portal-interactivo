Descripción general CanvasContext

NombreDescripción
CanvasContext.addColorStopAdd a circular gradient point.
CanvasContext.arcDibujar arco.
CanvasContext.beginPathComience a crear una ruta, debe usar el relleno o el trazo para llenar o acariciar la ruta.
CanvasContext.bezierCurveToCrea la ruta de la curva de Bezier cúbica.
CanvasContext.clearRectBorrar el contenido en el rectángulo.
CanvasContext.clipEstablezca la ruta creada como ruta recortada.
CanvasContext.closePathCerrar un camino.
CanvasContext.createCircularGradientCree un punto de gradiente circular, el punto de inicio es el centro del círculo y el punto final es el anillo.
CanvasContext.createLinearGradientCrear un gradiente lineal.
CanvasContext.drawDibuja la descripción en el contexto como el ruta, el estilo al canvas.
CanvasContext.drawImageDibuja la imagen y la imagen mantiene la información de tamaño original.
CanvasContext.fillLlena la ruta actual.
CanvasContext.fillRectLlena el rectángulo.
CanvasContext.fillTextComplete el texto en canvas.
CanvasContext.getImageDataObtenga los datos de píxeles del área implícita de canvas.
CanvasContext.lineToAgregue un nuevo punto y alinee el último punto especificado al nuevo punto.
CanvasContext.measureTextMida el tamaño del texto, la corriente solo se devuelve la bruja, y es una interfaz sincrónica.
CanvasContext.moveToMueva la ruta al punto especificado, la línea no se creará.
CanvasContext.putImageDataDibuje los datos de píxeles en canvas.
CanvasContext.quadraticCurveToCree una ruta de curva bezier cúbica cuadrática.
CanvasContext.rectDibuja un rectángulo.
CanvasContext.restoreRestaurar el contexto guardado.
CanvasContext.rotateEstablezca el punto original como el centro, gire la coordenada en el sentido de las agujas del reloj. El ángulo se agregará si gira se invoca varias veces.
CanvasContext.saveGuardar el contexto de canvas.
CanvasContext.scaleDespués de invoca la escala, la coordenada horizontal y vertical de la ruta creada posteriormente se escalará. La escala se multiplicará si la escala se invoca varias veces.
CanvasContext.setFillStyleEstablezca el color del relleno.
CanvasContext.setFontSizeEstablezca el tamaño de fuente.
CanvasContext.setGlobalAlphaEstablezca el alfa de Global Brush.
CanvasContext.setLineCapEstablezca el estilo de punto final de una línea.
CanvasContext.setLineDashEstablezca el estilo de la línea de tablero.
CanvasContext.setLineJoinEstablezca el estilo de la junta de las líneas.
CanvasContext.setLineWidthEstablezca el ancho de la línea.
CanvasContext.setMiterLimitEstablezca la longitud máxima de inglete.
CanvasContext.setShadowEstablezca el estilo de sombra.
CanvasContext.setStrokeStyleEstablezca el estilo de trazo.
CanvasContext.setTextAlignEl texto Alinee la propiedad utilizada para describir el texto en la API de Canvas 2D.
CanvasContext.setTextBaselineLa línea de base del texto actual utilizado para describir el texto en la API de Canvas 2D.
CanvasContext.setTransformRestablecer la transformación e invocar la transformación por matriz unitaria, la transformación se describe por la variable del método.
CanvasContext.strokeDibuja el trazo, por defecto esnegro.
CanvasContext.strokeRectDibuja un rectángulo sin relleno.
CanvasContext.toDataURLObtenga la URL de datos del área especificada del lienzo.
CanvasContext.toTempFilePathExporte el lienzo a una imagen y devuelva la ruta del archivo.
CanvasContext.transformUn método en el que una matriz se superpone varias veces en la transformación actual, y la matriz se describe mediante los parámetros del método. Puede escalar, rotar, mover e inclinar el contexto.
CanvasContext.translateTraslada el punto original de la coordenada. El punto original por defecto es la esquina superior izquierda.
