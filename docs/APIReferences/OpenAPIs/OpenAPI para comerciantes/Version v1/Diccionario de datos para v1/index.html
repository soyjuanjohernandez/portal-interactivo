PaymentFactor​

PropiedadTipo de datosRequeridaDescripción

OpenLoginIdInfo​

PropiedadTipo de datosRequeridaDescripción
loginIdString No

La ID de inicio de sesión es una identificación para un usuario, que puede ser un número de teléfono móvil o correo electrónico.El usuario puede usar ID de inicio de sesión para inicio de la billetera. Max.Longitud: 64 caracteres.

loginIdTypeString Yes

Tipo de ID de inicio de sesión, Enum: "MOBILE_PHONE", "EMAIL". Max.Longitud: 64 caracteres.

maskLoginIdString No

ID de inicio de sesión de máscara, se ocultarán varios bits del número de teléfono para proteger la privacidad de los usuarios. Max. length: 64 characters.

hashLoginIdString No

ID de inicio de sesión hash.El ID de inicio de sesión hahed por algoritmo hash.El sistema externo puede usarlo para compararEs ID de inicio de sesión hash por el mismo algoritmo hash para ver si la ID de inicio de sesión es la misma. Max.Longitud: 256 caracteres.

extendInfoString No

La información extendida, la billetera y el comerciante pueden poner información extendida aquí. Max.Longitud: 4096 caracteres.
