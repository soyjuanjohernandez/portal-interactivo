CanvasContext

CanvasContext

📄️ Descripción general CanvasContext

Nombre

📄️ CanvasContext.addColorStop

Agregue un punto de gradiente circular.

📄️ CanvasContext.arc

Dibujar arco.

📄️ CanvasContext.beginPath

Comience a crear una ruta, debe usar el `relleno` o el `trazo` para llenar o acariciar la ruta.

📄️ CanvasContext.bezierCurveTo

Crea la ruta de la curva de Bézier cúbica.

📄️ CanvasContext.clearRect

Borra el contenido del rectángulo.

📄️ CanvasContext.clip

Establezca la ruta creada como ruta recortada.

📄️ CanvasContext.closePath

Cerrar un camino.La ruta de cierre conectará el punto de inicio y el punto final. Si `fill()` o `stroke()` no se invoca y se crea una nueva ruta después de `closePath`, el camino anterior no se representará.

📄️ CanvasContext.createCircularGradient

Cree un punto de gradiente circular, el punto de inicio es el centro del círculo y el punto final es el anillo.Se debe llamar a `addColorStop()` para especificar el punto de gradiente y se deben especificar al menos dos puntos.

📄️ CanvasContext.createLinearGradient

Crear un gradiente lineal. Se debe llamar a `addColorStop()` para especificar el punto de gradiente y se deben especificar al menos dos puntos.

📄️ CanvasContext.draw

Dibuje la descripción en el contexto como el ruta, el estilo a canvas.

📄️ CanvasContext.drawImage

Dibuja la imagen y la imagen mantiene la información del tamaño original.

📄️ CanvasContext.fill

Llena la ruta actual.El color predeterminado es negro.

📄️ CanvasContext.fillRect

Llena el rectángulo.

📄️ CanvasContext.fillText

Llenar el texto en canvas.

📄️ CanvasContext.getImageData

Obtenga los datos de píxeles del área implícita de canvas.

📄️ CanvasContext.getImageData

Agregue un nuevo punto y alinee el último punto especificado al nuevo punto. `stroke()` se puede usar para dibujar la línea.

📄️ CanvasContext.measureText

Mida el tamaño del texto, la corriente solo se devuelve la bruja, y es una interfaz sincrónica.

📄️ CanvasContext.moveTo

Mueva la ruta al punto especificado, la línea no se creará. `stroke()` se puede usar para dibujar la línea.

📄️ CanvasContext.putImageData

Dibuja los datos de píxeles en canvas.

📄️ CanvasContext.quadraticCurveTo

Cree una ruta de curva bezier cúbica cuadrática. El punto de inicio del punto anterior del camino.

📄️ CanvasContext.rect

Dibuja un rectángulo. Usar `fill()` o `stroke()` para dibujar el rectángulo a canvas.

📄️ CanvasContext.restore

Restaurar el contexto guardado.

📄️ CanvasContext.rotate

Establezca el punto original como el centro, gire la coordenada en el sentido de las agujas del reloj.El ángulo se agregará si gira se invoca varias veces.

📄️ CanvasContext.save

Guardar el contexto del canvas.

📄️ CanvasContext.scale

Una vez invocada la `escala`, se escalarán las coordenadas horizontal y vertical de la trayectoria creada posteriormente. La escala se multiplicará si se invoca la `escala` varias veces.

📄️ CanvasContext.setFillStyle

Establezca el color del relleno.

📄️ CanvasContext.setFontSize

Establezca el tamaño de fuente.

📄️ CanvasContext.setGlobalAlpha

Establezca el alfa de Global Brush.

📄️ CanvasContext.setLineCap

Establezca el estilo de punto final de una línea.

📄️ CanvasContext.setLineDash

Establezca el estilo de la línea de tablero.

📄️ CanvasContext.setLineJoin

Establezca el estilo de la junta de las líneas.

📄️ CanvasContext.setLineWidth

Establezca el ancho de la línea.

📄️ CanvasContext.setMiterLimit

Establece la longitud máxima de inglete, que es la distancia entre los ángulos interior y exterior en la intersección de las dos líneas. Sólo es válido cuando `setLineJoin()` es `inglete`. Cuando se supera la longitud máxima, la unión se muestra como lineJoin como bisel.

📄️ CanvasContext.setShadow

Establezca el estilo que incluye la ubicación del color para la sombra.

📄️ CanvasContext.setStrokeStyle

Establezca el estilo de trazo. Por defecto, es negro.

📄️ CanvasContext.setTextAlign

Align es una propiedad de la API Canvas 2D que describe la alineación del texto cuando se dibuja. La alineación se basa en el valor x del método CanvasRenderingContext2D.FillText. Si textAlign="center", el texto se dibujará en 'x-50%*width'.

📄️ CanvasContext.setTextBaseline

Establezca las propiedades de la línea de base del texto actual.

📄️ CanvasContext.setTransform

Restablece la transformación e invocar la transformación por matriz unitaria, la transformación se describe por la variable del método.

📄️ CanvasContext.stroke

Dibujar el trazo de la trayectoria.

📄️ CanvasContext.strokeRect

Dibuja un rectángulo sin relleno.

📄️ CanvasContext.toDataURL

Obtener la URL de datos del área especificada de canvas.

📄️ CanvasContext.toTempFilePath

Exportar el lienzo a una imagen y devolver la ruta del archivo.

📄️ CanvasContext.transform

Un método en el que una matriz se superpone varias veces en la transformación actual, y la matriz se describe mediante los parámetros del método. Puede escalar, rotar, mover e inclinar el contexto.

📄️ CanvasContext.translate

Traducir el punto original de la coordenada.El punto original predeterminado es la esquina superior izquierda.
