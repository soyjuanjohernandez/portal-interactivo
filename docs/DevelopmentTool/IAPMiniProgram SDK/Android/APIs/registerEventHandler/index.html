registerEventHandler

La API de registerEventHandler es utilizado por la Súper Aplicación para registrar el controlador de eventos SDK.

Method signature​

void registerEventHandler(GriverEventManifest manifest, GriverContainerAPICallBack callBack)

Parámetros de solicitud​

NombreTipoDescripciónRequerido
manifestGriverEventManifestEl objeto del controlador de eventos.M
callBackGriverContainerAPICallBackLa callback que se utiliza para devolver el resultado del registro del controlador de eventos.O

Parámetros de respuesta​

NombreTipoDescripciónRequerido
errorErrorUn objeto que se usa para devolver el código de error y el mensaje de error cuando el registro falla.O

Códigos de error​

Código de errorMensaje de errorDescripciónNuevas medidas
90002GRV_CONTAINER_NOT_INITIALIZEDIAPMiniProgram SDKno se inicializa.Inicializar el SDK.
90003GRV_CONTAINER_ERROR_UNKNOWNError de parametro.Consulte la tabla de parámetros de solicitud y verifique si todos los tipos de parámetros son correctos y si se especifican todos los parámetros requeridos.
