# Dependencias de la biblioteca de terceros

El SDK iAs IApminiprogram utiliza múltiples bibliotecas de terceros para mejorar las funcionalidades y capacidades.Estas dependencias se incluyen en la Super App a través de la dependencia de Pod y pueden encontrar conflictos de versión de la biblioteca con otros SDK o la Super App.Se recomienda probar a fondo las funcionalidades de su aplicación después de la integración.Si se encuentra con cualquier versión en conflicto con las bibliotecas de terceros, comuníquese con nuestro soporte técnico a <a href="mailto:overseas_support@service.alibaba.com.">overseas_support@service.alibaba.com</a>

Este tema presenta las bibliotecas de terceros utilizadas en el SDK iAs IAPminiprogram y sus respectivos requisitos de versión.
